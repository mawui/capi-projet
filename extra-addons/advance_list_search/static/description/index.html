Advanced List Search

Advanced list search is a convenient solution to ease the search process. There is no need to keep in mind all the column names - a sticky header is here for you.

  • Overview
  • Releases
Advanced list search is easy to activate/deactivate for every user
While activated, the selected filters in the search bar are applied to all system views
Depending on the field type the Advanced list search works differently
For Text fields the search is conducted by any character that is present in the value
For numeric fields, the search is carried out by the exact value, or you can switch to interval mode
For Selection/Relation fields, the search is conducted by all available values for relation selection not limited to the data in the records. Multiple values can be selected.
For date and datetime fields, the search is carried out by the exact value, or you can also switch to interval mode
For datetime fields, you can switch to search by date mode
Applied filters in the Advanced list search can be added to Favorites in order to save time in the future and enjoy all the advantages of search
Sticky header - frozes the head of the table
Within Advanced list search module, it can be managed in the Settings > General Settings > Advanced List Search
